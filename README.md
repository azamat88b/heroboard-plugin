# ⚠️ Deprecated — moved to rnm-dev/claude-plugins

The Heroboard Claude Code plugin now lives in RNM's marketplace (`rnm-plugins`).

## Install
```
/plugin marketplace add rnm-dev/claude-plugins
/plugin install heroboard@rnm-plugins
```

If you installed from this repo, remove it and reinstall from the marketplace above:
```
/plugin marketplace remove heroboard
```

This repo is frozen and no longer receives updates.
